Transition

Define which transition to use — slide or crossfade — using data-transition:

<div class="fotorama"
     data-transition="crossfade">
  <img src="1.jpg">
  <img src="2.jpg">
</div>

Dissolve

There is one more transition, dissolve. This variation of fade is perfect for demonstrating images with the identical fragments.

Dissolve example (new window):

Dissolve   Crossfade

Mix

Have one transition on swipe and another on click using data-clicktransition:

<div class="fotorama"
     data-transition="slide"
     data-clicktransition="crossfade">
  <img src="1.jpg">
  <img src="2.jpg">
</div>
